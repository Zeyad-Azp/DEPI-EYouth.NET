## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/DEPI-DotNet-Tasks.git
```

2. Open the project using Visual Studio or VS Code

3. Restore dependencies:

```bash
dotnet restore
```

4. Run the project:

```bash
dotnet run
```
